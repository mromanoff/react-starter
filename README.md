React Starter

---
 
Welcome.

+HMR is supported with `webpack-hot-middleware` and `react-hot-loader`. Updates to your React components will be reflected immediately in the browser.

Usage
---
 
Start the development server with this command:
 
```
npm start
```
 
 
Setup
---
 
```
npm install
```
 
 
Compile
---
 
```
npm run compile
```
